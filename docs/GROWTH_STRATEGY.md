> 작성일: 2026-04-05

---

## 1. 핵심 문제 인식

### 하객이 QR을 찍을 이유가 없다

현재 플로우:
```
QR 스캔 → SMS 인증 → 금액 입력 → 끝
```

하객 입장의 심리:
- 봉투 내고 끝인데 굳이 QR까지 찍어야 하나?
- SMS 인증이 번거롭다
- 내가 얻는 게 뭔지 모르겠다

**결론: 하객에게 명확한 이득이 없으면 참여율이 낮다.**

---

## 2. 해결책 - 하객 참여 유도

### 핵심 가치 제안
> "내가 낸 축의금 기록이 평생 남는다"

| 하객 이득 | 설명 |
|-----------|------|
| 영수증 보관 | 내가 낸 금액 디지털 증빙 |
| 답례 기준 | 나중에 답례 받을 때 기준점 |
| 역방향 조회 | 상대방 결혼 때 내가 얼마 받았는지 확인 |
| 히스토리 누적 | 앱 설치 후 평생 내 경조사 내역 조회 |

### 즉각적 보상 - 카카오톡 알림톡
축의금 입력 완료 시 카카오톡으로 영수증 발송:

```
[정담] 축의금 전달 완료

○○○님이 신랑·신부 결혼식에
30,000원을 축의하셨습니다 🎊

- 일시: 2026년 5월 1일
- 장소: ○○웨딩홀

정담 앱에서 내 축의 내역을 확인하세요.
[앱 다운로드]
```

## 3. 하객 → 호스트 전환 퍼널

```
하객이 QR 찍음
    ↓
SMS 인증
("내 기록을 저장하기 위해" 라는 맥락으로 납득)
    ↓
축의금 입력
    ↓
카카오톡 영수증 수신 (즉각적 보상)
    ↓
[나중에] 앱 설치 → 폰번호로 로그인
    ↓
내 전체 축의 히스토리 조회
    ↓
본인 결혼 준비 시 → 호스트로 전환
```

**이 퍼널이 핵심. 하객을 미래 호스트로 만드는 구조.**

## 4. 카카오 알림톡 연동 계획

### 사용 서비스
- **솔라피(Solapi)** - 현재 SMS에 이미 사용 중, 알림톡도 지원

### 연동 순서

| 단계 | 작업 | 소요 기간 |
|------|------|-----------|
| 1 | 카카오 비즈니스 가입 + 사업자 등록 | 1~3일 |
| 2 | 카카오톡 채널 개설 (정담 공식 채널) | 즉시 |
| 3 | 알림톡 템플릿 작성 → 카카오 심사 | 2~5일 |
| 4 | 솔라피 콘솔에서 카카오 채널 연동 | 즉시 |
| 5 | API route에서 솔라피 알림톡 API 호출 구현 | 1~2일 |

### 비용
- 알림톡: 건당 약 **₩8~10원**
- SMS 대비 저렴하고 오픈율 압도적으로 높음

### 알림톡 승인 전 임시 대응 (지금 바로 개발 가능)
→ 축의금 완료 후 **영수증 화면** 웹에서 표시  
→ 스크린샷 저장 / 카카오톡으로 공유 버튼 제공  
→ 비용 0원으로 동일한 UX 제공 가능

### 필요한 알림톡 템플릿 목록

| 템플릿명 | 발송 시점 | 수신자 |
|----------|-----------|--------|
| 축의금 전달 완료 | 하객 축의금 입력 완료 | 하객 |
| 조의금 전달 완료 | 하객 조의금 입력 완료 | 하객 |
| 새 축의금 알림 | 하객이 축의금 입력 시 | 호스트 |
| 방명록 작성 완료 | 방명록 제출 완료 | 하객 (선택) |

## 5. 비즈니스 모델 (BM)

### BM 우선순위

| BM 방식 | 내용 | 현실성 | 우선순위 |
|---------|------|--------|---------|
| **명부 실물 굿즈** | 디지털 명부 → 실물 책자 인쇄 배송 ₩29,900~₩49,900 | 높음 | ★★★ 1순위 |
| **프리미엄 청첩장 템플릿** | 고급 디자인 ₩9,900~₩19,900 | 중간 | ★★ 2순위 |
| **B2B 웨딩홀 제휴** | 웨딩홀에 월 구독료 | 가장 높음 | ★ 3순위 (장기) |

### 알림톡 비용 구조
```
현재: 비용 0원 (영수증 화면 스크린샷으로 대체)
    ↓
프리미엄 플랜 출시 시:
    - 알림톡 영수증 발송 포함
    - 비용을 호스트 구독료에 전가
    - 건당 ₩10원 × 하객 수 → 소규모 행사 기준 ₩1,000~5,000원
```

### 명부 굿즈 상세 기획 (1순위)
```
디지털 명부 → 실물 책자로 인쇄

포함 내용:
- 커버: 커플 이름, 날짜, 장소 레이아웃
- 내지: 하객 이름, 금액, 관계, 메시지 정리
- 마무리: QR 코드 (디지털 버전 링크)

유통 방법:
- 앱에서 "실물 명부 주문하기" 버튼
- 정보 자동 정리 → 인쇄소 연동 → 배송
- 가격: ₩29,900~₩49,900 (행사 완료 후 주문)
```

## 8. SMS → 알림톡 마이그레이션 계획

현재 Twilio SMS를 솔라피로 전환 여부 검토:

| 항목 | Twilio | 솔라피 |
|------|--------|--------|
| SMS 단가 | 높음 | 낮음 |
| 알림톡 지원 | 없음 | 있음 |
| 국내 특화 | 낮음 | 높음 |
| API 난이도 | 쉬움 | 쉬움 |

→ **솔라피로 통합 전환 권장**  
→ `pages/api/send-verification.js` Twilio 로직을 솔라피로 교체
